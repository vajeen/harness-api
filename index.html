Chucknorris API

Chucknorris API for OneFlow is a JSON API for hand curated Chuck Norris facts.

Usage

Retrieve list of all chuck jokes in JSON format.

http://chucknorris.home.lab/list

Example response:

        
            {

                "id" : "9D-5xHGrRpiegWjjV4fRLg",

                "joke" : "Chuck Norris keeps a grenade launcher hidden inside his grand piano."

            },

            {

                "id" : "jXl332D3Ttmmn-eraYC9iA",

                "joke" : "When Chuck Norris smiles, someone dies. When he's smiling while roundhouse kicking someone, then two people die."

            }

Retrieve a random chuck joke in JSON format.

http://chucknorris.home.lab/rand

Example response:

        
            {

                "id" : "9D-5xHGrRpiegWjjV4fRLg",

                "joke" : "Chuck Norris keeps a grenade launcher hidden inside his grand piano."

            }

Retrieve a chuck joke by ID.

http://chucknorris.home.lab/joke/9D-5xHGrRpiegWjjV4fRLg

Example response:

        
            {

                "id" : "9D-5xHGrRpiegWjjV4fRLg",

                "joke" : "Chuck Norris keeps a grenade launcher hidden inside his grand piano."

            }
This API was developed for OneFlow as an assignment and not for commercial use.
